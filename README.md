# React Native RemixIcon
### Open Source Library for Open Source Icons

## Installation
```bash
npm install react-native-remixicon
```

## Usage
```jsx
import RemixIcon from 'react-native-remixicon';

<RemixIcon name="macbook-line" size={24} color="#000" />
```

## Props
| Name | Type | Default | Description |
| --- | --- | --- | --- |
| name | string | - | Icon name |
| size | number | 24 | Icon size |
| color | string | #000 | Icon color |
| ... | any | - | Other props |
